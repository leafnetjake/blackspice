# blackspice

A theme for [Vim](https://www.vim.org/) and [Neovim](https://neovim.io/).

## Install

To install using [vim-plug](https://github.com/junegunn/vim-plug), add this line to your `vimrc` or `init.vim`:

```vim
Plug 'leafnetjake/blackspice'

To later use it add this line to your `init.vim` or `.vimrc` file:
colorscheme blackspice
```
